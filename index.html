Tecnologías

html

Es el lenguaje que se utiliza para estructurar y organizar el contenido de una página web. Define elementos como textos, imágenes, enlaces y secciones, permitiendo crear la base sobre la que se construye un sitio.

css

Es el lenguaje de estilo que da forma y diseño a una página web. Permite modificar colores, fuentes, márgenes y distribuir los elementos para que la interfaz sea atractiva y fácil de usar.

javascript

Es un lenguaje de programación que agrega interactividad y dinamismo a las páginas web. Gracias a él, los sitios pueden responder a las acciones del usuario y actualizar contenido sin recargar la página.

node.js

Es un entorno que permite ejecutar JavaScript fuera del navegador. Se utiliza principalmente para crear servidores y desarrollar aplicaciones del lado del backend, haciendo que JavaScript pueda usarse tanto en el cliente como en el servidor.
